tmux常用命令以及快捷键

发表于 2019-03-04

命令行

tmux new[-session] -s $session_name -d 新建会话(-d 是否在后台)
tmux new -s name -n $window_name 新建会话及窗口
tmux at[tach] -t session 重新连接(-t 后接会话名称)
tmux ls 显示保存的会话
tmux kill-session -t $session_name 删除会话
